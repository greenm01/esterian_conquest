# Release Bundles

This directory holds demo-ready Esterian Conquest zip packages for emulator testing and operator reproduction.

Current bundles:

- `ec-v1.5-classic-demo.zip`: original packed DOS executables plus a known-good local-console `CHAIN.TXT`
- `ec-v1.5-unlocked-demo.zip`: unlocked plain-MZ executables in the same layout and with the same `CHAIN.TXT`

Both archives are generated by:

```bash
python3 scripts/build_release_packages.py --verify
```

The bundled game directory is built from the preserved `fixtures/ecutil-init/v1.5` baseline so package generation does not depend on mutable working copies under `original/v1.5/`.
